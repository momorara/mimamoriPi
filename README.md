# mimamoriPi
mimamori for Pi

<h4><<概要>></h4>
Raspberry Pi(ラズパイ)を使った見守り機能基板のためのプログラムです。<br>
人感センサーにより人を感知して、ambientへのデータを投げてのダッシュボード表示などを行います。<br>
さらにAHTx0を使用して気温、湿度を測定します。<br>
気温、湿度から簡易的にＷＢＧＴをを計算して表示します。<br>

それらのデータはwifiを使いambientという計測値表示サービスに投げていますので、表示データ保存が可能となっています。<br>
*参考サイト==>https://ambidata.io/bd/board.html?id=76151 <br>
これらは、基本無料サービスの範囲で使用できるものであり、 見守り機能を無料で実現できます。 ただし、通信についてはwifi(or 有線LAN)を利用する必要があります。<br>
また、これらサービスを無料で使うためにそれぞれのサービスにご自身で 登録する必要があります。<br>
　・LEDの色等指定はできません。<br>
　・部品の仕様が変わる場合があります。 <br>
　・基板のバージョンが変わる場合がありますが、機能等に違いはありません。<br>

 用途としては、<br>
・ひとり親の簡易見守り<br>
・子供の簡易見守り<br>
・無人建物の簡易監視<br>
・ペットの簡易見守り<br>
が考えられます。<br>

ハードウエア的になるべく安価にできるように最小構成としています。<br>
ただし、必要最低限の機能は有しています。<br>
また、表示器のOLEDとスイッチ2個の増設が可能なパターンを基板に作っているので、ご自身で増設しpythonプログラムを追加カスタマイズすることが可能となっています。<br>
